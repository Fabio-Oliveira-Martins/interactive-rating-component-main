## Sobre

Este projeto foi um desafio fornecido pela plataforma Frontend Mentor que consistia em desenvolver um formulário de avaliação. O formulário deveria conter 5 botões numerados de 1 a 5 — referenciando o nível de satisfação — e um botão para enviar a nota. Após o envio da nota, deveria aparecer uma tela de agradecimento com a nota fornecida pelo usuário. O principal objetivo desta aplicação era desenvolver habilidades e familiaridade com a criação de formulários e com a interação entre o site e o usuário, através de uma interface agradável e intuitiva.

### Requisitos do desafio:

- Respeitar o layout original;
- Conter 5 botões numerados de 1 a 5;
- Conter 1 botão para envio do formulário;
- Permitir interação entre usuário e o formulário;
- Mostrar tela de agredecimento com a nota fornecida pelo usuário;
- Layout responsivo;
- Incluir estados visuais (hover, focus...).


**Este programação também utiliza métodos de acessibilidade para navegação com teclado.**

### Navegação manual

1º - Copie e cole o link abaixo na barra de pesquisa do seu navegador:


2º - Usando o mouse, selecione com o botão esquerdo um dos cinco botões numerados dentro do formulário. Quando um deles estiver selecionado, o mesmo ficará da cor branca destacando qual opção foi selecionada.

3º - Após escolhida a nota, pressione o botão escrito "SUBMIT" para enviar a nota.

4º - Por fim, você verá uma tela de agradeciemento mostrando a nota que você deu, e depois de 5 segundos a página irá retornar ao formulário.

1º - Copie e cole o link abaixo na barra de pesquisa do seu navegador:

2º - Usando o teclado, pressione a tecla TAB para selecionar o primeiro botão que contém a nota 1, ao redor do botão aparecerá um circulo indicando que o mesmo está selecionado.

3º - Para selecionar as outras opções:
- **Mover para esquerda**: pressione a tecla com uma seta para cima ou para esquerda.
- **Mover para direita**: pressione a tecla com uma seta para baixo ou para direita.

4º - Após selecionar o botão com a nota, pressione o botão escrito "SUBMIT" para enviar a nota.

5º - Por fim, você verá uma tela de agradeciemento mostrando a nota que você deu, e depois de 5 segundos a página irá retornar ao formulário.
